Características del framework

Ember CLI Ember-cli aims to bring convention over configuration to build tools.
Ember Data Most Ember applications use Ember Data, a data-persistence library providing many of the facilities of an object-relational mapping (ORM).
Ember Inspector The Ember Inspector is an extension currently available for the Mozilla Firefox and Google Chrome web browsers for debugging Ember applications.
Fastboot Fastboot is an Ember CLI addon created by the Ember core team that gives Ember developers the ability to run their apps in Node.js.
Liquid Fire Liquid Fire provides animation support for Ember applications.

Para ver ejemplos, consultar el blog oficial del framework.
