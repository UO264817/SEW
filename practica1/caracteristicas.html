Características del framework

Característica Descripción
Ember CLI Ember-cli pretende llevar la convención sobre la configuración a las herramientas de construcción.
Ember Data La mayoría de las aplicaciones Ember utilizan Ember Data, una biblioteca de persistencia de datos que proporciona muchas de las facilidades de un mapeo objeto-relacional (ORM).
Ember Inspector El Ember Inspector es una extensión disponible actualmente para Mozilla Firefox y Google Chrome para la depuración de aplicaciones Ember.
Fastboot Fastboot es un complemento de Ember CLI creado por el equipo del núcleo de Ember que ofrece a los desarrolladores de Ember la capacidad de ejecutar sus aplicaciones en Node.js.
Liquid Fire Liquid Fire proporciona soporte de animación para las aplicaciones Ember.

Para ver ejemplos, consultar el blog oficial del framework.
